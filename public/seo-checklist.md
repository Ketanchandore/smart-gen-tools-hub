## üåê DOMAIN & HOSTING SETUP

### Hostinger Setup Steps:
1. Purchase domain and hosting from Hostinger
2. Point domain to Hostinger nameservers
3. Upload website files to public_html folder
4. Set up SSL certificate (usually automatic)
5. Configure DNS records if needed

### Google Search Console Setup:
1. Go to search.google.com/search-console
2. Add your domain property
3. Verify ownership using HTML file method or DNS
4. Submit sitemap: https://yourdomain.com/sitemap.xml
5. Monitor indexing status
